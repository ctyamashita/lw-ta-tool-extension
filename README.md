## Installation
### 1. Clone this repo
### 2. On your chrome-based browser open:
  - Extensions (`chrome://extensions`)
  - Enable Developer Mode
  - Load Unpacked
  - Select the repo folder
  - Enable extension if disabled
## Basic functions
- Hide Wott chats from tickets page
- Turn yourself back on after break
- Collect and view tickets from a batch

## Tickets page
- you can change the displayed batch by passing `?batch=<BATCH_NUMBER>` on the end of the url

## Notes
- If tickets page stops showing any info try clearing and collecting data again
